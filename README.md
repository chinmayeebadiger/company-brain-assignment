# Company Brain - Architecture Research Assignment

## Objective

The goal of this assignment is to:

1. Learn the complete GitHub contribution workflow.
2. Understand different components of the Company Brain architecture.
3. Research technologies used
4. Practice collaboration through pull requests.

---

## Assignment Overview

Each participant must:

1. Pick one component from the architecture.
2. Research a technology that could be used to implement that component.
3. Document their findings.
4. Submit the work through GitHub using a proper contribution workflow.

---

## Available Components

Choose **ONE** component from the list below.

Examples:

- Vector Database
- Graph Database
- Event Queue / Message Broker
- Search Engine
- Frontend Dashboard
- Async Workers
- LLM Layer
- Caching Layer

---

## Step 1 - Fork the Repository

Fork this repository to your GitHub account.

Clone your fork locally:

```bash
git clone <your-fork-url>
```

---

## Step 2 - Create a Branch

Create a branch named:

```bash
feature/<your-name>-<component>
```

Example:

```bash
feature/chinmayee-vector-db
```

---

## Step 3 - Create Your Submission

Inside the `research` folder create:

```text
research/<your-name>.md
```

Example:

```text
research/chinmayee.md
```

---

## Step 4 - Complete the Research

Use the following template:

```markdown
# Component

Vector Database

# Proposed Technology

Qdrant

# What does this component do?

Explain the purpose of the component.

# Why did you choose this technology?

- Reason 1
- Reason 2
- Reason 3

# Advantages

- Advantage 1
- Advantage 2

# Limitations

- Limitation 1
- Limitation 2

# Would you use it in Company Brain?

Explain your decision.

# References

Add useful links.
```

---

## Step 5 - Commit Your Work

Commit using a meaningful message.

Example:

```bash
git commit -m "Add Qdrant research proposal"
```

---

## Step 6 - Push Changes

Push your branch to GitHub.

Example:

```bash
git push origin feature/chinmayee-vector-db
```

---

## Step 7 - Open a Pull Request

Open a Pull Request to the main repository.

PR Title Format:

```text
[Research] <Technology Name>
```

Example:

```text
[Research] Qdrant
```

---

## Deliverables

To complete the assignment you must:

- Claim an issue
- Create a branch
- Submit research
- Open a Pull Request
